# DevLog 2025-12-29
## 2025-12-29 개발 일지: PointTutor 핵심 기능 개선 및 확장

**요약:** PointTutor 프로젝트에 인증 기능 추가, 녹음 모드 개선, PDF 캡처 기능 향상을 통해 사용자 경험을 전반적으로 개선했습니다. 특히 인증 기능 추가로 보안성을 강화하고, 녹음 모드와 PDF 캡처 기능의 효율성을 높여 사용자 편의성을 증대시켰습니다.

**주요 작업:**

*   **✨ 인증 기능 추가:** 사용자 계정 관리 및 보안 강화를 위한 기본적인 인증 기능을 구현했습니다. 로그인/로그아웃 기능과 간단한 회원 가입 기능을 포함합니다.
*   **♻️ 녹음 모드 개선:** 녹음 품질 및 안정성을 향상시키고, 사용자 인터페이스를 개선하여 더욱 직관적인 녹음 경험을 제공합니다. 기존 녹음 과정에서 발생하던 몇 가지 버그들을 수정했습니다.
*   **🐛 PDF 캡처 기능 개선:** PDF 문서의 특정 영역을 캡처하는 기능의 정확도를 높이고 성능을 최적화했습니다. PDF 파일 크기가 클 경우 발생하던 문제점을 해결했습니다.

**기술적 도전:**

*   인증 기능 구현 시, 보안 취약점을 최소화하기 위해 다양한 보안 라이브러리 및 모듈을 검토하고 적용하는 과정에서 시간과 노력이 많이 소요되었습니다. 특히 암호화 알고리즘 선택에 심혈을 기울였습니다.
*   PDF 캡처 기능 개선 과정에서 PDF 문서 구조의 복잡성으로 인해 원하는 영역을 정확하게 추출하는 로직을 구현하는 데 어려움이 있었습니다.

**다음 계획:**

*   인증 기능에 소셜 로그인 기능을 추가하여 사용자 편의성을 더욱 높일 예정입니다.
*   녹음 모드에 노이즈 제거 기능을 추가하여 녹음 품질을 더욱 향상시킬 계획입니다.
*   PDF 캡처 기능을 활용하여 사용자가 캡처한 영역을 편집하고 저장할 수 있는 기능을 개발할 예정입니다. 또한, 더 다양한 형식의 문서 포맷을 지원할 수 있도록 확장할 계획입니다.
