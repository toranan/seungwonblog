# DevLog 2025-12-30
## PointTutor: PDF 영역 캡처 및 음성 컨트롤 기능 추가

**요약:** PointTutor 프로젝트에 PDF 영역 캡처 기능과 음성 컨트롤 기능을 추가하여 사용성을 향상시켰습니다. 이를 통해 사용자는 PDF 문서의 특정 영역을 더욱 쉽고 정확하게 캡처하고, 음성 명령을 통해 PointTutor를 제어할 수 있게 되었습니다.

**주요 작업:**

*   ✨ **b7cad76 - PDF 영역 캡처 개선:** PDF 문서 내에서 원하는 영역을 정확하게 캡처할 수 있도록 캡처 기능을 개선했습니다. 기존 캡처 방식의 부정확성을 해결하고, 사용자 인터페이스를 개선하여 사용성을 높였습니다.
*   ✨ **b7cad76 - 음성 컨트롤 기능 추가:** 음성 명령을 통해 PointTutor를 제어할 수 있는 기능을 추가했습니다. 이를 통해 마우스나 키보드 사용 없이도 기본적인 기능을 실행할 수 있도록 편의성을 높였습니다.

**기술적 도전:**

*   PDF 문서의 다양한 구조와 포맷을 안정적으로 처리하면서 정확한 영역을 캡처하는 알고리즘 개발에 어려움이 있었습니다.
*   다양한 사용자의 발음과 억양을 인식하고 정확하게 명령을 수행하는 음성 인식 모델을 구축하는 데 어려움이 있었습니다. 음성 인식 API의 제한사항과 성능 문제도 고려해야 했습니다.

**다음 계획:**

*   캡처 기능의 정확도를 더욱 향상시키고, 다양한 PDF 렌더링 엔진과의 호환성을 높일 예정입니다.
*   음성 컨트롤 기능에 더 많은 명령어를 추가하고, 사용자 정의 명령어를 설정할 수 있도록 기능을 확장할 계획입니다. 또한, 사용자 피드백을 바탕으로 음성 인식 정확도를 개선할 예정입니다.
